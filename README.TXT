=Description=

This is a ToDoList built in Java that allows users to add, remove, or change elements in a partial array.
-------------------------------------------------------------------------------------------------------
=How To Use=

First enter how many tasks you would like in the ToDoList. A menu is then shown in the console as it 
is shown below.

Select Activity below:
        [A]dd a Task
        [R]emove a Task
        [C]hange a Task
        [Q]uit
